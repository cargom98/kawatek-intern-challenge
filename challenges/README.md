# Kawatek Internship Technical Challenge 2027

Welcome to the Kawatek technical evaluation. This challenge is designed to assess your problem-solving skills, code quality, and alignment with the work you'll do at Kawatek.

## About Kawatek

Kawatek Co., Ltd. is a Japanese deep-tech company focused on intelligent bionic systems, robotics, Physical AI, rehabilitation technologies, and human augmentation. We develop the RYO® bionic hand, AI-powered rehabilitation platforms, and next-generation robotic systems.

## How to Complete This Challenge

1. **Fork** this repository
2. Choose the challenge that matches your internship track:
   - `challenge-a/` → Physical AI & Robotics Interns
   - `challenge-b/` → AI Programming & UX Interns
3. Complete the tasks described in the challenge README
4. Push your solution to your fork
5. Submit the link to your fork via email to the provided address

## Evaluation Criteria

| Criteria | Weight |
|----------|--------|
| Correctness & functionality | 30% |
| Code quality & organization | 25% |
| Problem-solving approach | 20% |
| Documentation & explanation | 15% |
| Creativity & bonus features | 10% |

## Rules

- You have **5 days** from the date you receive this challenge
- You may use any online resources, documentation, or AI tools — but you must be able to **explain your solution** in a follow-up interview
- Include a `SOLUTION.md` in your challenge folder explaining your approach
- Commit history matters — we want to see your thought process, not a single giant commit

## Questions?

If you have questions about the challenge, open an issue in this repo.

Good luck! 🚀
